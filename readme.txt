Allows Postmatic subscriptions via Gravity Forms

== Description ==
This simple plugin allows individuals to subscribe to Postmatic feeds using Gravity Forms.

== Installation ==
1. Download the zip.
2. Extract the contents into your wp-content/plugins folder.
3. Activate.

== Frequently Asked Questions ==
How do I configure my form?

1. Access your form.
2. Hover over _Settings_ at the top of the page and click on _Postmatic_.
3. Create a new feed.
4. Configure the _Email_ and _Name_ settings to correspond with the appropriate form fields.
5. If desired, use conditional logic to control if a user is subscribed or not.
